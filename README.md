### 👩‍💻 Como contribuir
#### 1. Clone o repositório do projeto
```
$ git clone https://github.com/IzaqueNunes/audio-app.git
```
#### 2. Acesse a pasta
```
$ cd audio-app
```
#### 3. Instale as dependencias
```
$ npm install
```
#### 4. Rode o app
```
$ npx expo start
```
